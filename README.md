# Red Queen Emulator

A full-stack emulator project with a static frontend and Python Flask backend.

## Quick Start

### Frontend (static)

```bash
npm install
npm run dev
```
Visit http://localhost:3000/

### Backend (Flask REST API)

```bash
cd backend
pip install -r requirements.txt
python app.py
```
Visit http://localhost:5000/

## How it works

- Frontend: public/index.html, styles/main.css, src/main.js
- Backend: Flask API in backend/app.py
- CORS enabled for local dev

---

## Deploy to Production

- Host frontend on Vercel, Netlify, GitHub Pages, etc.
- Deploy backend with PyPI host, Heroku, etc.
